Название проекта: интернет-магазин  видео игр "Historical games"
Пуликация проекта: https://blooming-hollows-30065.herokuapp.com
Код проекта: https://github.com/pavelusov79/django_site_games

Функционал проекта:
- регистрация пользователей с подтверждением почтового ящика
- авторизация
- добавление товаров в корзину авторизованным пользователям, редактирование товаров в корзине
- оформление заказов, редактирование и удаление заказов (при редактировании и удалении заказов кол-во товаров соответственно меняется)
- реализация товаров со скидкой
- реализация поиска по товарам и новостям
- добавление подписчиков через форму
- отправка писем через форму обратной связи на сайте
- реализована кастомная админка проекта

Вход в админку через форму авторизации. 

Логин админа: django
Пароль: geekbrains

Возможности админки:
- crud операции по пользователям
- crud операции по товарам и категориям
- удаление подписчиков
- crud операции новостей

Несколько страниц было сверстано по макету psd (главная страница, страница каталога, страница отдельного товара), остальные страницы с использованием bootstrap. Верстка адаптивная.
